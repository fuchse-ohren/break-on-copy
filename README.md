# Break on Copy

Break on Copy は、Webページ上で選択した文字列をショートカットでコピーし、
**連続する空行を自動で整形**してクリップボードへ保存する Firefox 向け拡張です。

## 主な機能

- 任意のショートカットで選択テキストをコピー
- 2つ以上連続した改行を1つに圧縮
- コピー後に選択範囲を解除
- マウス位置の近くにコピー結果の簡易トーストを表示
- `iframe`（同一オリジンのみ）内の選択にも対応
- ポップアップ画面でショートカット設定を保存

## 動作イメージ

1. ページ内の文字列を選択する
2. 設定したショートカットキーを押す（初期値: **Shift + C**）
3. 整形済みテキストがクリップボードへコピーされる

## 整形ルール

コピー時に以下の整形を行います。

- `\r\n` / `\r` / `\n` が2回以上続く箇所を `\n` 1つに置換
- 文字列の先頭・末尾の空白を `trim()` で除去

## 設定項目

ポップアップ（拡張アイコンをクリック）から以下を設定できます。

- 組み合わせキー
  - Shift
  - Ctrl
  - Alt
  - Meta
- 英字キー（1文字）

設定値は `browser.storage.local` の `copy_key_settei` に保存されます。

## ファイル構成

- `manifest.json` : 拡張のマニフェスト
- `selcopy.js` : コンテンツスクリプト本体（コピー処理・イベント監視）
- `options/options.html` : 設定画面
- `options/options.js` : 設定画面のロジック
- `options/options.css` : 設定画面のスタイル
- `icons/icon.png` : 拡張アイコン

## 開発・動作確認（Firefox）

1. Firefoxで `about:debugging#/runtime/this-firefox` を開く
2. **一時的なアドオンを読み込む** を選択
3. このリポジトリの `manifest.json` を指定
4. 任意のページでテキストを選択し、ショートカットを押して動作確認
