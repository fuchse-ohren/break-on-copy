# Break on Copy

Break on Copy は、Webページ上で選択した文字列をショートカットでコピーし、
**連続する空行を自動で整形**してクリップボードへ保存する Firefox 向け拡張です。

## 主な機能

- `manifest.json` の `commands` で定義したショートカットを使用
- `browser.commands.onCommand` でコマンドを受け取り、選択テキストを整形コピー
- 2つ以上連続した改行を1つに圧縮
- コピー後に選択範囲を解除
- マウス位置の近くにコピー結果の簡易トーストを表示

## 動作イメージ

1. ページ内の文字列を選択する
2. ショートカットキーを押す（初期値: **Shift + C**）
3. 整形済みテキストがクリップボードへコピーされる

## 整形ルール

コピー時に以下の整形を行います。

- `\r\n` / `\r` / `\n` が2回以上続く箇所を `\n` 1つに置換
- 文字列の先頭・末尾の空白を `trim()` で除去

## ショートカット設定

ショートカットの変更は Firefox 標準の管理画面で行います。

1. `about:addons` を開く
2. 歯車メニューから **拡張機能のショートカットを管理** を開く
3. `Break on Copy` のショートカットを変更する

## ファイル構成

- `manifest.json` : 拡張のマニフェスト（コマンド定義を含む）
- `background.js` : コマンド受信とタブへのメッセージ送信
- `selcopy.js` : コンテンツスクリプト本体（コピー処理・トースト表示）
- `options/options.html` : 使い方表示用のポップアップ
- `options/options.css` : ポップアップのスタイル
- `icons/icon.png` : 拡張アイコン

## 開発・動作確認（Firefox）

1. Firefoxで `about:debugging#/runtime/this-firefox` を開く
2. **一時的なアドオンを読み込む** を選択
3. このリポジトリの `manifest.json` を指定
4. 任意のページでテキストを選択し、ショートカットを押して動作確認
